##  **HOW TO USE THIS:**

### **Method 1: GitHub Web Interface**
1. Go to your GitHub repository
2. Click **"Add file"** → **"Create new file"**
3. In the filename box, type: `requirements.txt`
4. Copy and paste all the text above
5. Scroll down and click **"Commit new file"**

### **Method 2: If Working Locally**
1. Create a new file named `requirements.txt` in your project folder
2. Paste the content above
3. Save the file

##  **WHAT EACH LIBRARY DOES:**

| Library | Purpose | Your Usage |
|---------|---------|------------|
| **torch** | PyTorch deep learning framework | Model training & inference |
| **transformers** | Hugging Face library for BERT models | DistilBERT fine-tuning |
| **datasets** | Dataset loading and processing | Handling training data |
| **pandas** | Data manipulation | CSV processing |
| **numpy** | Numerical operations | Array operations |
| **scikit-learn** | ML metrics and preprocessing | Accuracy, precision, recall |
| **gradio** | Web UI framework | Interactive web app |
| **accelerate** | Training optimization | Mixed precision (FP16) |
| **tqdm** | Progress bars | Training progress display |
| **matplotlib** | Plotting (optional) | Visualizations |
| **seaborn** | Statistical plots (optional) | Advanced visualizations |
